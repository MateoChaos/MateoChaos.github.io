Benvinguts als Projectes

Em dic Mateo Chaos soc alumne de robótica de 1r de Batxillerat de l'Institut Pompeu Fabra

Aquesta pàgina está escrita en HTML (llenguatge de marques d'hipertext) que dona l'estructura física (paràgrafs, títols, llistes, imatges, links, videos, audio) emprant etiquetes.i utilitza CSS (fulles d'estil en cascada) que ens ajuda a donar la forma i l'estil a la web

A més utilitzen JavaScript que proporciona tota la interactivitat de la pàgina web.

Utilitza biblioteques externes com bootstrap que conté JavaScript i CSS i que ens permet fer un menú desplegable i posar un logo fácilment.

Una altra llibreria externa que utilitzem és Font Awesome que es una biblioteca que ens permet posar símbols amb facilitat, son els símbols que estan en el menú.
