Variant la intensitat d'un led: utilitzant pulsador i PWM de l'arduino

Podem veure una simulació que hem creat una variable anomenada step que té un valor de 17 unitats perquè es multiple de 255 i fara cada vegada que pulsem un cop augmenta l'altre botó disminueix l'intensitat de la llum que tindrá a veure amb la intensitat de r, g i b o el sistema de color. Crec que farem servir (HSL o HSV), que es basa en la tonalitat del color o hue

void setup()
{
  pinMode(13, OUTPUT);
}

void loop()
{
  digitalWrite(13, HIGH);
  delay(1000); // Wait for 1000 millisecond(s)
  digitalWrite(13, LOW);
  delay(1000); // Wait for 1000 millisecond(s)
}
